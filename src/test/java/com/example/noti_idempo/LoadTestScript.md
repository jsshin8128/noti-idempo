# 부하 테스트 가이드

## JUnit 테스트 실행

```bash
./gradlew test --tests LoadTest
```

테스트 결과는 터미널에 출력됩니다. (`build.gradle`에 `showStandardStreams = true` 설정됨)

## 테스트 설정

- 동시 요청: 200 스레드
- 스레드당 요청: 20회
- 총 요청: 4,000건
- 사용자 ID: 1~10 순환
- 알림 유형: LIKE
